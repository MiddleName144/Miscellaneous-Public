In order to change scripts to functional keyboard shortcuts, follow this short list of steps:
1. Open the desired script in a text editor.
2. Change lines 28 and 29 to your email and password.
3. Convert to a .exe file with auto-python-to-exe.
4. Turn it into a shortcut (right-click, new, shortcut)
6. Put the .exe shortcut file onto your desktop.
7. In properties, change 'window type' to 'minimized', grant administrator privileges, and set shortcut key to whatever function you would like.
Do note that this is specifically for logging in from google.com
